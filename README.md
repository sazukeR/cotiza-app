## How to use it

1. Clone the repository

```
git clone https://github.com/sazukeR/cotiza-app.git
```

2. Move to project folder

```
cd cotiza-app
```

3. Install project dependencies

```
yarn
```

4. Run the project

```
yarn dev
```
